## GetZLib で Z-Library を探す

GetZLib は、Z-Library 公式サイト、Z-Library 最新リンク、Z Library
最新アドレス、マジックメールの使い方、Z-Library アプリ ダウンロードを
探す方のための独立した情報ガイドです。古いドメインを探し回る代わりに、
[GetZLib 日本語版](https://getzlib.com/ja)で現在のアクセス情報と実用ガイドを
まとめて確認できます。

## Z-Library の最新サイトとアドレスを確認する

Z-Library のアドレスは変更されることがあるため、静的な README に保存した
リンクはすぐ古くなる可能性があります。[GetZLib ホームページ](https://getzlib.com/ja)
では、最新サイトの情報と利用可能なアクセス方法を案内しています。ページを
ブックマークし、現在の Z-Library アドレスを確認したいときにご利用ください。
